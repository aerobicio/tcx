[TCX](http://developer.garmin.com/schemas/tcx/v2/) File Parser.


## Installation

Add this line to your application's Gemfile:

    gem 'tcx', :git => 'git://github.com/quamen/tcx.git'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tcx
